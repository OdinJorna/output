Vanuit deze wiki: https://benwebber.github.io/tiddlywiki-dnd/# heb ik een knop gekopieerd en aangepast. Deze voegt een notitie toe (wat een losse eigen tiddler is). Die notitie houdt in de list field bij bij welke tiddler die hoort. Op de tiddler worden de inhoud van de notie bovenaangeplaatst.

Ik heb de Notitie tag toegevoegd aan shit die niet geexporteerd moet worden. Zowel in het build command, als in de public versie van de node explorer. Dus ze zijn hidden voor players

  • $:/benwebber/notes/dnd/images/add-note
  • $:/benwebber/notes/dnd/stylesheet/notes
  • Add note
  • $:/benwebber/notes/dnd/ui/ViewTemplate/notes

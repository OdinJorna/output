Wiki
  • Ik heb zelf een pagecover 'plugin' geschreven voor deze wiki. De plugin was te groot en onhandig voor wat ik precies wilde.
    • de field page-cover zorgt ervoor dat de tiddler in viewmode een foto krijgt. Plaats een URL in het field als link
    • de field page-cover-class is ervoor om optionele CSS classes toe te voegen. Tot zo ver heb ik de volgende classes:
      1. pc-10 om de foto is hoger te plaatsen
      2. pc-30 om de foto is lager te plaatsen

Alle tiddlers voor deze plugin beginnen met Pagecover/. Hier is een lijst:

  • Pagecover/image height
  • Pagecover/set image height
  • Pagecover/stylesheet
  • Pagecover/view template

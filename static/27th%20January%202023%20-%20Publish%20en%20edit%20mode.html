27th January 2023 - Publish en edit mode

 27th January 2023 at 12:38pm 
Wiki

Omdat ik deze wiki wil kunnen exporteren naar iets wat de spelers ook kunnen gebruiken, heb ik een publish en edit mode toggle gemaakt. Je vind dit in de Sidebar/Admin panel, ook in de sidebar.

Ik moet niet alleen bij het exporteren, alle tiddlers met tag Geheim, Gereedschap en Wiki wegfilteren. Ik moet ook de node explorer aanpassen, zodat backlinks er niet nog steeds in voorkomen. En de kapstok veranderen, zodat geheime shit niet in hun inhoudsopgave komt.

Node explorer

Dit doe ik door twee node explorers te hebben. Door middel van de toggle in het admin panel, wordt de ViewTemplate tag heen en weer geswitched. De publisch variant is hetzelfde als de oorspronkelijke, maar ik heb de volgende drie regels toegevoegd aan het MainFilter():

-[tag[Geheim]]
-[tag[Gereedschap]]
-[tag[Wiki]]

Node explorer tiddlers:

  • $:/plugins/kookma/viewtemplates/node-explorer
  • $:/plugins/kookma/viewtemplates/node-explorer-details
  • $:/plugins/kookma/viewtemplates/node-explorer-publish
